🐾 Animal Cards Project
Интерактивный сайт о животных, созданный с использованием Tailwind CSS и JavaScript. Проект демонстрирует работу с модальными окнами и современными UI-паттернами.

✨ Основные возможности
📱 Адаптивный дизайн — корректное отображение на всех устройствах.

🎨 Tailwind CSS — быстрая и аккуратная стилизация без лишнего кода.

⚡ JavaScript — интерактивные элементы и модальные окна для карточек животных.

🖼️ Карточки питомцев — каждая карточка открывается в модальном окне с подробной информацией.

🔧 Логичная структура проекта — легко поддерживать и расширять.

🚀 Цель проекта
Создать удобный и визуально приятный сайт, где пользователи могут познакомиться с животными и узнать больше о каждом из них.

🛠️ Технологии
HTML

Tailwind CSS

JavaScript

📂 Структура проекта
index.html — главная страница

style.css — стили (Tailwind)

script.js — логика модальных окон и интерактивных элементов
